# ShortURL Frontend

Frontend cho dự án ShortURL – ứng dụng rút gọn link và phân tích lượt click. Được xây dựng bằng React + Vite + TailwindCSS.

## 🚀 Tính năng
- Giao diện đơn giản để nhập URL dài và nhận URL ngắn
- Hiển thị thống kê lượt click từ backend
- Responsive UI, tối ưu cho cả desktop và mobile
- Tích hợp với backend qua REST API

## ⚙️ Cài đặt & chạy
Clone repo: git clone https://github.com/HoangNghiaHoa/shortURL.git && cd url-shortener-fe  
Cài dependencies: npm install  
Chạy dev server: npm run dev  
Ứng dụng sẽ chạy tại http://localhost:5173

## 📦 Build production
npm run build  
Output sẽ nằm trong thư mục dist/

## 🔑 Biến môi trường
Tạo file .env ở root với nội dung:  
VITE_API_URL=http://localhost:8080
